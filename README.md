# API em Python com FastAPI

## Descrição

Este projeto consiste no desenvolvimento de uma **API REST** criada inteiramente em **Python** utilizando o framework **FastAPI**.  
A aplicação foi construída com o objetivo de fornecer uma estrutura moderna, rápida e organizada para criação de rotas, manipulação de dados e comunicação entre sistemas.

A FastAPI foi escolhida por sua alta performance, simplicidade de uso e geração automática de documentação interativa, facilitando tanto o desenvolvimento quanto os testes da aplicação.

---

## Objetivo

O principal objetivo deste projeto é implementar uma API robusta e de fácil manutenção, aplicando boas práticas de desenvolvimento back-end com Python.

Entre os objetivos da aplicação, destacam-se:

- criar endpoints para comunicação entre cliente e servidor
- organizar a estrutura do projeto de forma clara
- facilitar testes das rotas
- permitir integração com front-end ou outros sistemas
- explorar o uso do FastAPI no desenvolvimento de APIs modernas

---
